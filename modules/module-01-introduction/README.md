# Module 1: Introduction to n8n and Workflow Automation

## Overview

This module introduces you to n8n, a powerful workflow automation platform, and teaches you the fundamentals of workflow automation. By the end of this module, you'll be able to set up n8n and create your first workflows.

**Duration:** 1 week (10-15 hours)

## Learning Objectives

- Understand what n8n is and how it compares to other automation tools
- Set up your n8n environment
- Navigate the n8n interface
- Create your first simple workflow

## Key Concepts Covered

- **Workflow Automation**: Connecting different applications and services to automate repetitive tasks
- **Nodes**: Building blocks of workflows that perform specific actions
- **Triggers**: Events that start a workflow execution
- **Executions**: Individual runs of a workflow
- **Credentials**: Stored authentication information for services

## Prerequisites

- Understanding of web applications and APIs (helpful but not required)
- Text editor installed on your system
